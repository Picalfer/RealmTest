# Realm Compose Test

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)
[![Realm](https://img.shields.io/badge/Database-Realm_SDK-purple.svg)](https://www.mongodb.com/docs/realm/sdk/kotlin/)

Тестовое приложение для изучения Realm Database с Jetpack Compose в Android. Демонстрирует сложные связи между объектами и реактивные запросы.

## Функциональность

- Отображение курсов с преподавателями и студентами
- Диалог с детальной информацией о курсе
- Удаление курсов из базы данных
- Реактивные запросы с Flow
- Сложные связи между моделями (one-to-many, backlinks)

## Технологии

- Kotlin
- Jetpack Compose
- Realm Database
- ViewModel
- Coroutines + Flow
